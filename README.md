# Kaggle House Prices Prediction Solution

This repository contains my machine learning project solving the popular Kaggle "House Prices - Advanced Regression Techniques" competition. 

## Overview & Approach
The goal of this project is to predict house sales prices based on various property features. I extracted key structural data points to train a predictive model.

### Features Used:
* **LotArea:** Size of the lot in square feet
* **YearBuilt:** Original construction date
* **1stFlrSF & 2ndFlrSF:** First and second-floor square footage
* **FullBath:** Full bathrooms above grade
* **BedroomAbvGr & TotRmsAbvGrd:** Total bedrooms and rooms above grade

##  Machine Learning Model
* **Model:** Random Forest Regressor (`RandomForestRegressor`)
* **Framework:** Scikit-Learn (`sklearn`)
* **Visualization:** Added a Seaborn distribution plot tracking the variation in output prices generated by the final model.

##  How to Run
1. Open the `house_prices_random_forest.ipynb` notebook.
2. Ensure you have `pandas`, `matplotlib`, and `seaborn` installed.
3. Run all cells to process the Ames housing test data and see the prediction trend graph.
